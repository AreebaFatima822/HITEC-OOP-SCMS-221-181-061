## Project Description

The Smart Campus Management System (SCMS) is an Object-Oriented Programming project developed in C++. The system aims to automate and manage various campus-related activities such as student records, course management, faculty information, attendance, and academic data. The project is being developed using OOP principles to ensure modularity, reusability, and maintainability. Additional features and modules will be added as the development process continues.



## OOP Concepts Demonstrated

The following OOP concepts will be implemented in this project:

1. Classes: All modules are implemented as classes. 
2. Objects: Objects represent students, faculty, books, etc.
3. Encapsulation: Data members are kept private with public methods.
4. Abstraction: Complex operations are hidden behind interfaces.
5. Inheritance: Specialized classes inherit from base classes.
6. Single Inheritance: Student inherits from Person.
7. Multilevel Inheritance: Derived classes inherit from one base class.
8. Multiple Inheritance: Combined functionality from multiple parents.
9. Hierarchical Inheritance: Multiple classes inherits from one base class. 
10. Polymorphism: Same interface used for different objects.
11. Function Overloading: Methods perform similar tasks with different parameters.
12. Operator Overloading: Operators customized for class objects.
13. Function Overriding: Derived classes redefine base classes. 
14. Virtual Functions: Runtime polymorphism implementation.
15. Pure virtual functions: Abstract classes enforce implementation.
16. Abstract Classes: Generalized parent classes. 
17. Constructors: Initiliaze object data.
18. Destructors: Release recourses automatically.
19. Copy constructor: Copy object data safely. 
20. Friend function: Controlled access to private data.
21. Static Members: Shared information among objects.
22. Dynamic Memory Allocation: Runtime memory management using new/delete.
23	Exception Handling: Handles invalid operations and errors.

24	File Handling: Persistent storage of records.

25	STL Containers: Vectors, maps, and lists used for data management.
